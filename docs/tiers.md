# Fanatic Bingo — User Tier Structure

## Guest (No Account)
- One free game per device, tracked by cookie
- Multiplayer guest sessions (invited by a host) do NOT consume the free game
- After the free game, user is prompted to create a free account
- If the user exits the game without creating an account, and their free game has been used, the next time they sign in the first page they see is the create an account page. User cannot move forward until they create an account. 
- If a user signs in for the first time via multiplayer the first screen they should see is the login/ create an account screen. 

## Free Account
- Unlimited games at the current feature set
- Can see premium features (Create Your Own, league filters, Blackout Bingo) but they are grayed out
- No cost — requires email registration only

## Pass (Single Purchase)
- Temporary access to premium features: 3-hour, 6-hour, 24-hour, or weekend options
- Access to Create Your Own board
- Custom terms/categories submitted via Create Your Own are NOT eligible for consideration in the main term library (premium-only feature)
- Access to league filters and Blackout Bingo for the duration of the pass

## Premium (Subscription)
- Full access to all features
- Create Your Own board with custom term submission for consideration in the main library
- League filters (major/minor league dropdowns)
- Blackout Bingo mode (available after winning traditional Bingo)
- Any future premium features
- Vote on adding and removing terms

## Dev / Builder Bypass
- A dev username/password bypass for internal testing
- Grants full premium access without payment
- Must not be accessible to general users

## All Users
- Ask if user wants username to be remembered
- For mulitplayer ask if group name wants to be rememebered
